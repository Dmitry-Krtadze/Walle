#  Проєкт WALL-E (ESP32 + Arduino Mega)

Цей проєкт реалізує дворівневу систему керування роботом WALL-E, де **ESP32** виступає як "мозок" (зв'язок, звук, вебсервер), а **Arduino Mega** — як "тіло" (керування моторами, сенсорами та анімацією).

---

## 🛠 Останні технічні зміни та покращення

### 1. Автономна логіка (Сонар)
* [cite_start]Додано постійний моніторинг відстані за допомогою ультразвукового датчика **HC-SR04** (Trig: 12, Echo: 13)[cite: 53].
* Якщо перешкода знаходиться ближче ніж **30 см** протягом **5 секунд**, робот автоматично активує відтворення випадкового аудіофайлу з SD-карти.

### 2. Система "Живих емоцій"
* [cite_start]**Голова (Серво):** Під час відтворення аудіо голова рухається у випадкових напрямках із рандомним прискоренням, імітуючи реакцію на звук[cite: 53].
* **Очі (RGB LED):** Замість простого блимання реалізовано плавне перетікання кольорів (ефект градієнта) через функцію `nblend`.
* [cite_start]**Автостоп:** Після завершення треку робот повертає голову в центр (90°) та вимикає підсвітку очей[cite: 54, 102].

### 3. Вебінтерфейс (Українською)
* Повністю перекладено інтерфейс керування.
* Додано слайдер регулювання швидкості моторів (команда `V`).
* Реалізовано "геймпадну" логіку кнопок: робот їде тільки поки кнопка затиснута.

## 🔌 Інструкція з перепайки підсилювача PAM8302A

Для стабільної роботи звуку безпосередньо з ESP32, модуль підсилювача має бути підключений наступним чином:

| Пін підсилювача | Куди підключати | Примітка |
| :--- | :--- | :--- |
| **Audio IN+** | **GPIO 25** (ESP32) | Вихід вбудованого ЦАП (DAC1) |
| **Audio IN-** | **GND** | Спільна земля сигналу |
| **VIN / VCC** | **5V** | Живлення для максимальної гучності |
| **GND** | **GND** | Земля живлення |
| **Shutdown (SD)**| **5V / VIN** | **Важливо:** припаяти перемичку, щоб вивести модуль з режиму сну |

## 📋 Таблиця підключень компонентів

### Зв'язок між платами (UART)
* **ESP32 TX2 (17)** ➔ **Arduino Mega RX3 (15)**
* **ESP32 RX2 (16)** ➔ **Arduino Mega TX3 (14)**
* **Спільна земля (GND)** має бути об'єднана між обома платами.

### Периферія (Arduino Mega)
* [cite_start]**Мотори (A):** 9, 8 [cite: 53]
* [cite_start]**Мотори (B):** 7, 6 [cite: 53]
* [cite_start]**Сонар:** Trig (12), Echo (13) [cite: 53]
* [cite_start]**Серво:** Пін 2 [cite: 130]
* [cite_start]**LED Очі:** Пін 42 [cite: 53]
* **Дисплей:** I2C (SDA/SCL)
